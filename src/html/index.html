Lua

Introdução

Este é a documentação do Game Lua, O qual foi desenvolvido utilizando C++ juntamente com Lua, utilizando somente ferramentas livres, e podendo ser alterado e comercializado desde que aceite a licensa prevista no projeto.

Histórico

Lua game se encontra na versão 0.1 em fase de desenvolvimento o qual tem sido criado para desenvolvimento de Trabalho de Conclusão de Curso de Sistemas de Informação na UNIOURO - Ouro Preto do Oeste ainda em 2009 para ser apresentado no final deste ano;

Ferramentas Utilizadas

Este projeto faz uma integração de duas linguagens de programação que são: C++ - Para o motor do jogo; Lua - Para os scripts de configuração, extensão e controle

Para a Documentação usamos o Doxygen. SDL para recursos de Vídeos e Eventos.
